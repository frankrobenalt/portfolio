Goal Bank

Currently converting from Angular.js to React Native
Github Repo
Goal Bank is an application that I built using Angular.js, Node, Express, and PostgreSQL. Users create a profile that keeps track of goals that they set for themselves. Users set wagers in the beginning and if they don't hit meet their goals, the wagers go to a charity of their choice. This was my first full-stack application, so I got to learn how web applications work from front-end to back-end. Through this project, I learned the importance of writing reusable code and the importance of planning before writing code. This was an incredible learning experience with plenty of roadblocks.
